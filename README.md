# Lobe Chrome Extension with React and Typescript

> Lobe Chrome extension that hits Lobe's local API. 

## Features 

- Project Selection
- Image inference
- Add to Dataset

## Development

### Install

```sh
npm install
```

### Usage

```sh
npm run watch
```

### Build

```sh
npm run build
```

### Load in Browser
- Go to Extensions for browser (Edge/Chrome)
- Load Unpacked
- Select `dist` directory
